# RastreiaCartas
O que é? Aplicativo em PWA que administra coleção de cartas do jogo HearthStone. O aplicativo é simples e fácil de utilizar, servindo para ajudar o usuário a lembrar de seu deck, possibilitando criar algumas novas opções em qualquer lugar. O aplicativo será capaz de deixar o usuário determinar também a porcentagem de vítoria do jogador com o deck atual dele.

Como baixar? Para baixar e utilizar como um aplicativo, basta acessar o link no navegador do seu celular, clicar nas configurações da aba e clicar em alguma opção parecida com "Baixar aplicativo".

Descricao de cada pagina/tela:

Na pagina inicial: mostra as cartas que estão selecionadas.Tem a opção de Adicionar uma nova carta a partir do botão de mais (+)
Tem a opção de remover uma carta do deck atual caso 
clique na carta e então no botão de lixo.

Ainda na primeira tela o usuário poderá acompanhar sua % de vitória adicionando seus jogos.

Na Segunda tela: ao clicar no botão de mais o usuário é levado para a segunda tela, é exibido a coleção do usuário e caso ele clique em uma carta ele poderá adicionala ao seu deck efetivo na primeira pagina.
